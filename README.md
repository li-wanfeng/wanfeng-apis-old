# Spring Boot初始项目模板

#### 介绍
 **基于比较新的 springboot2.7 所创建的后台项目模板，快速搭建你的项目** 

#### 软件架构
基于mvc架构设计，进行了分层


#### 主流框架

1.  Spring Boot 2.7.14
2.  Spring MVC
3.  MyBatis Plus持久层框架（已配置好分页）
4.  Spring AOP

#### 数据存储

1.  mysql数据库 8.0.26
2.  待更新
3.  ....

#### 业务特性

1.  全局异常处理器
2.  自定义错误码
3.  封装通用响应类
4.  Knife4j 接口文档
5.  基于注解+AOP实现的自定义鉴权
6.  全局跨域处理
